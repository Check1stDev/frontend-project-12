# Chat (Slack)

Веб-приложение для командного общения в формате Slack. Пользователи могут обмениваться сообщениями, работать с каналами и взаимодействовать с интерфейсом в реальном времени.

## Возможности

- регистрация и авторизация пользователей;
- создание, удаление и переименование каналов;
- обмен сообщениями;
- переключение между каналами;
- обработка ошибок и уведомления;
- адаптивный интерфейс.

## Технологии

- React
- Vite
- JavaScript
- Axios
- React Router
- Bootstrap
- Render
